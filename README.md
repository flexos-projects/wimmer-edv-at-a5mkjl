# Wimmer Consulting Website

Welcome to the official website for Wimmer Consulting.

## Description
Wimmer Consulting ist Ihr strategischer IT-Partner in Österreich, spezialisiert auf die digitale Transformation mittelständischer Unternehmen. Wir liefern maßgeschneiderte Lösungen in den Bereichen ERP-Systeme (Microsoft Dynamics 365), Cloud-Infrastruktur, IT-Sicherheit und individuelle Softwareentwicklung, um Ihre Geschäftsprozesse zu optimieren und Ihre Wettbewerbsfähigkeit zu steigern.

## Tech Stack
This website is built with:
*   **Astro**: A modern static site builder for fast content-driven websites.
*   **TypeScript**: For enhanced developer experience and type safety.
*   **Tailwind CSS**: A utility-first CSS framework for rapidly building custom designs.

## Getting Started

### Prerequisites
Make sure you have Node.js (v18 or higher) and npm installed.

### Installation
1.  **Clone the repository:**
    ```bash
    git clone [your-repo-url]
    cd wimmer-consulting-website
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```

### Running Locally

1.  **Start the development server:**
    ```bash
    npm run dev
    ```
    This will start a local server, usually at `http://localhost:4321`.

2.  **Build for production:**
    ```bash
    npm run build
    ```
    This command compiles the project for production, outputting static assets to the `dist/` directory.

3.  **Preview the production build:**
    ```bash
    npm run preview
    ```
    This command serves the `dist/` directory locally, allowing you to preview the production build before deployment.
