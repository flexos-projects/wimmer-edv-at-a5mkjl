# Pages & Navigation

> This document outlines the website's structure, including the sitemap, navigation hierarchy, and details for each individual page.

## Sitemap & Navigation Structure

The website follows a logical, hierarchical structure designed for intuitive user navigation and efficient search engine crawling.

**Primary Navigation:**
- **Services** (Dropdown)
  - ERP (Microsoft Dynamics 365)
  - Cloud & IT Infrastruktur
  - IT-Sicherheit
  - Softwareentwicklung
  - Legacy Modernisierung (AS/400)
- **Case Studies**
- **Über Uns**
- **Blog**
- **Kontakt** (Primary CTA Button)

**Sitemap Hierarchy:**
```
/
├─ services/
│  ├─ erp-dynamics-365/
│  ├─ cloud-infrastruktur/
│  ├─ it-sicherheit/
│  ├─ softwareentwicklung/
│  └─ legacy-modernisierung/
├─ case-studies/
├─ ueber-uns/
├─ blog/
│  └─ [slug]/
├─ kontakt/
└─ impressum/
└─ datenschutz/
```
